| **Pulmonary Arterial Pressure** (PAP) | 15-30 mmHg Systolic (PSAP)

HR x SV/1000 | 4-8 LPM |
| **Cardiac Index (CI)**  
CO/BSA | 2.5-4 LPM/m3 |

Diastolic BP - PAOP | 60-80 mmHg |

SV/EDV x 100 | 40-60% |

SV/EF | 45-90 mL/m2 |

CO/HR x 1000 | 60-100 mL/beat |
| **Stroke Volume Index** (SVI)

SVmax-SVmin/SVmean x 100 | 10-15% |

| **Left Ventricular Stoke Work Index** (LVSWI)  
SI x (MAP - PAOP) x 0.0136 | 50-62 g/m2/beat |
| **Right Ventricular Stoke Work** (RVSW)  
SI x MAP x 0.0144 | 51-61 g/m/m2 |
| **Right Ventricular Stoke Work** Index (RVSWI)  
SVI x (MPAP -CVP) x 0.0136 | 5-10 g/m2/beat |
