### Antibiotic Redosing Protocols

**_Warning:_** _The tables below are example “Antibiotic Re-dosing Protocols” from a university hospital’s manual for anesthesia providers._  
_Institutional protocols should be followed.  
_  
**Pre-operative: (Infusion to incision time)**  
If > 60 minutes from start of antibiotic infusion to incision, patient should be re-dosed.  
For vancomycin or ciprofloxacin: If > 120 minutes from start of infusion to incision, patient should be re-dosed.  
  
**Procedures > 4 hours (normal renal function)**

**Antibiotic Re-dosing Tables with renal function included (CrCl=Creatinine Clearance)**  
  
**Ampicillin/sulbactum (Unasyn** ® **)**

table.tableizer-table { font-size: 12px; border: 1px solid #CCC; font-family: Arial, Helvetica, sans-serif; } .tableizer-table td { padding: 4px; margin: 3px; border: 1px solid #CCC; } .tableizer-table th { background-color: #104E8B; color: #FFF; font-weight: bold; }

| Pre-operative dose: 3 grams IV over 15-30 minutes |
| --- |
| CrCl > 50 mL/min: Re-dose Q 2 hours with the same pre-op dose |
| CrCl 10-50 mL/min: Re-dose Q 4 hours with the same pre-op dose |
| CrCl < 10 mL/min: Not recommended |
| Total postoperative doses: 3 |
