# Nipride (Nitroprusside) 400mcg/mL  
in mcg/kg/min

  
100mg in 250ml (400mcg/mL or 0.4mg/mL)  
Common Range 0.5-5mcg/kg/min

* * *

**Emergency Stick**  
With a 10cc syringe-draw up 1mL from the above IV bag(200mcg/mL) and add  
9mL = 20mcg/mL  
Follow with a prepared flush syringe

* * *

**Equation (mcg/kg/min)**  
Desired dose x kg x 60 / mcg/mL = mL per hr

* * *

**Example**  
you desire 0.5mcg/kg/min for a 70k pt  
0.5 x 70 x 60 / 400 = mL/hr  
2,100 / 400 = 5.25 so 5.25mL/hr = 0.5mcg/kg/min

* * *

  
**You should always start at 0.25mcg/kg/min**
