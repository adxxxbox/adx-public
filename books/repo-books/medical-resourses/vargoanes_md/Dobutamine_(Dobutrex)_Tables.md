# Dubutamine  
4,000mcg/cc

### Dobutamine (Dobutrex)

  
**Equation**  
Dobutamine (Dobutrex) 1,000mg/250ml Bag(4,000mcg/cc)  
Desired Dose x kg x 60 / 4,000 = ml/hr

* * *

**Example:**  
Order of 5mcg/kg/min for a 70 kg pt=  
(5 x 70) x 60= 21,000/4,000  
\= 5.25 ml per hr for 5mcg/kg/min
