# Nitroglycerine (Tridil) Tables in mcg/kg/min

### 25mg/250 mL bottle (100mcg/mL)

  
Mostly titrate to effect because there is not real an optimal dose in mcg/kg/min.  
Initial dose of 0.2mcg/kg min  
**Range:** 0.2 - 1.5 mcg/kg/min, increasing increments of 0.1 mcg/kg/min Q 3-5 mins.  
**Desired dose x kg x 60 / mcg/mL = mL per hr**

* * *

**Example:**  
If you desire 0.5mcg/kg/min for a 70k pt  
0.5 x 70 x 60 / 100 = mL/hr  
2100 / 100 = 21 so 21mL/hr = 0.5mcg/kg/min

* * *
