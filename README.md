# adx-public


To require these functions in your userscript, use the following format:
```
// @require https://raw.githubusercontent.com/adxxxbox/adx-public/refs/heads/main/functions/{file_name}.js

```

So far available functions are:
- power-menu-injecter.js: Utilities for adding custom items to ChatGPT's power popup menu. To use, require as follows:
```
// @require https://raw.githubusercontent.com/adxxxbox/adx-public/refs/heads/main/functions/power-menu-injecter.js
```
- modal-utils.js: Utilities for creating and managing modal dialogs in ChatGPT. To use, require as follows:
```
// @require https://raw.githubusercontent.com/adxxxbox/adx-public/refs/heads/main/functions/modal-utils.js
```
- storage-utils.js: Utilities for managing persistent storage in userscripts. To use, require as follows:
```
// @require https://raw.githubusercontent.com/adxxxbox/adx-public/refs/heads/main/functions/storage-utils.js
```
